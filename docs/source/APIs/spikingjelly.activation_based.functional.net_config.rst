Network Configuration Functions
++++++++++++++++++++++++++++++++++++

这些函数帮助用户统一设置网络中每个 **子模块的配置** ，如步进模式、后端等。

----

These functions help users set **configurations for each submodule** in a network, such as step mode and backend.
