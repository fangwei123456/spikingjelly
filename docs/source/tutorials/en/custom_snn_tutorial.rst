.. _custom_snn_tutorial-en:\n\nCustom SNN Tutorial / 自定义 SNN 教程\n===================================\n\n本教程介绍如何在 SpikingJelly 中构建自定义 SNN 层。\nThis tutorial introduces how to build custom SNN layers in SpikingJelly.\n\n1. 介绍 (Introduction)\n----------------------\n\n在本节中，我们将演示如何实现一个简单的自定义神经元。\nIn this section, we will demonstrate how to implement a simple custom neuron.\n\n.. code-block:: python\n\n    import torch\n    from spikingjelly.activation_based import neuron\n\n    class CustomNeuron(neuron.BaseNode):\n        def __init__(self, v_threshold=1.0):\n            super().__init__(v_threshold=v_threshold)\n            self.v_threshold = v_threshold\n\n        def neuronal_charge(self, x: torch.Tensor):\n            self.v = self.v + x\n\n        def neuronal_fire(self):\n            return self.surrogate_function(self.v - self.v_threshold)\n\n        def neuronal_reset(self, spike):\n            if self.v_reset is None:\n                self.v = self.v - spike * self.v_threshold\n            else:\n                self.v = (1. - spike) * self.v + spike * self.v_reset\n
